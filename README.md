# Yatube API

Социальная сеть Yatube — это платформа для публикации постов, комментариев и подписок. Проект предоставляет REST API для взаимодействия с основными сущностями: посты, комментарии, группы, подписки.

## 📋 Возможности

- Аутентификация пользователей (JWT-токены)
- Создание, просмотр, редактирование и удаление постов
- Комментирование постов
- Просмотр списка сообществ (групп)
- Система подписок на авторов
- Пагинация (limit/offset) и поиск по подпискам
- Разграничение прав: только автор может изменять или удалять свой контент

## 🛠️ Технологии

- Python 3.10+
- Django 3.2
- Django REST Framework (DRF)
- Simple JWT
- SQLite (по умолчанию)
